## Phase 1 react 粗筛 WebSearch query
1. `荣耀 MagicOS 分屏 手势 进入 滑动 两个窗口 内容 互换 交换`
   → 命中"一步手势分屏"功能（2024.05），官方支持页提到拖动长条分屏、上下/左右分屏切换。
2. `荣耀 MagicOS 分屏 两个应用 位置 调换 拖动 交换 上下分屏 左右分屏 切换`
   → **关键命中**：荣耀官方支持页"长按分屏窗口顶部的横条至分屏窗口缩小后，拖拽该窗口至另外一个分屏窗口，可实现分屏互换"。
3. `荣耀 "长按分屏窗口" 横条 缩小 拖拽 "另外一个分屏窗口" 互换 honor.com`
   → 确认互换原文同时出现在荣耀(honor.com)与华为(consumer.huawei.com)官方支持页。
4. `荣耀 分屏 进入 横向滑动 上下分屏 纵向滑动 左右分屏 手势 三指 上滑`
   → 命中"竖屏状态下用指关节横向划过屏幕；横屏状态下纵向划过屏幕""三指上滑启动分屏"等手势触发原文。

→ 有强信号，进入 Phase 2 深抓。

## Phase 2 react 深抓 WebFetch
- 荣耀官方支持页 `honor.com/cn/support/content/zh-cn15819407/`（"分屏，轻松应对多任务"）：WebFetch 返回 404；curl 下载得 824KB SPA 外壳（无内联正文，正文经 JS 异步加载），落盘 `honor-support-splitscreen.html`。实际正文以 WebSearch 摘要为准。
- 华为官方支持页 `consumer.huawei.com/cn/support/content/zh-cn15991539/`（同题"分屏，轻松应对多任务"）：进入分屏手势（底部上滑左上 / 边缘内滑调出智慧多窗）+ "长按分屏窗口顶部...拖拽该窗口至另外一个分屏区域，松手即可分屏互换"。
- 百度经验 `jingyan.baidu.com/article/fb48e8be203d042f632e1413.html`：荣耀分屏操作；更新日期 **2021-08-04**（晚于专利公开日 2017.02.08）。
- 华为官方"华为手机如何分屏" `consumer.huawei.com/cn/support/content/zh-cn15812820/`：指关节启动分屏"竖屏状态下，用指关节横向划过屏幕；横屏状态下，纵向划过屏幕"。
