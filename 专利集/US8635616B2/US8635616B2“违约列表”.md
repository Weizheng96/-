### 4.2 Patent license / open source pledge caveat（**关键**）

本专利的核心技术（vDPA = BE/FE + SR-IOV VF + DMA workflow）几乎完全对应 Linux kernel vDPA framework；如果华为已就 US8635616B2 向 Linux Foundation、OIN（Open Invention Network）、或类似 patent pledge 组织做出 commitment，则上游 Linux kernel + 商业发行版 + 商业 vendor 实施可能受 patent license 覆盖。

**法务必查**：
- 华为是否是 OIN 成员？OIN 包含哪些 vDPA 相关核心专利？
- 华为是否有任何 vDPA / virtio 相关 IPR statement？
- 华为是否有公开 patent pledge 用于推动 Linux 生态？

如有，所有"确认侵权"档候选 verdict 应进一步降级。

### 4.3 法律免责声明

本报告输出**线索与证据链**，**不构成法律意见**。本专利目前法律状态为 **Active（已授权 2014-01-21，预计失效 2032-05-22）**。

但本专利涉及 Linux kernel 主线 vDPA framework — 涉及成千上万下游使用方；维权前应仔细评估：
- Patent license / open source pledge 状态
- 反诉风险（Red Hat / NVIDIA / Intel 等的反向专利墙）
- 标准化 / 事实标准的合理实施抗辩

专利侵权最终判定由专业律师分析。
