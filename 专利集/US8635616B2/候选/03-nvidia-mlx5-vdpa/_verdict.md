# Verdict — NVIDIA mlx5_vdpa driver + BlueField/ConnectX

> S3+S4+S5；适用独立权：Claim 1 / 11 / 19 / 20 / 21；分级：**P0**

## 1. 核心组织
**NVIDIA Corporation（NASDAQ: NVDA）+ Mellanox Technologies 子公司**

## 2. F1-F9 命中表

NVIDIA 自家 documentation 公开:
- BlueField + ConnectX 系列 NIC 提供 SR-IOV VF + virtio-net acceleration（vDPA）
- mlx5_vdpa kernel driver 由 Mellanox/NVIDIA 工程师上游贡献到 Linux kernel
- DOCA SDK 含 vDPA examples

F1-F9 字面命中（NVIDIA 既是上游 vDPA 实现方又是商业卖家）

## 3. 时间线
- mlx5_vdpa 上游 first commits：2020+
- BlueField-2 GA：2020-08；BlueField-3 GA：2023
- 完全在 post-grant 时间窗内

## 4. §A 穿透
- §A.1 NVIDIA 在 G06F9 / H04L 专利墙；建议 IncoPat 补查
- §A.7 NVIDIA / Mellanox 邮箱域工程师在 Linux kernel vDPA 上游大量 commits（强归因证据）
- §A.4 NVIDIA 官方 BlueField SDK 文档 + DOCA examples

## 5. 状态机三栏：**第 1 档（确认侵权-高）**（NVIDIA 既是上游贡献方又商业卖卡）+ patent license caveat

## 6. 总结
NVIDIA mlx5_vdpa 既贡献上游又商业卖 BlueField / ConnectX with vDPA support；F1-F9 全部字面命中；落第 1 档（确认侵权-高）；patent license caveat 同候选 01。
