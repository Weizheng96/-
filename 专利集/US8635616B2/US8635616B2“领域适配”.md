# US8635616B2 领域适配

## 1. 主领域

**通信 / 云 / 网络**（具体子方向：**hypervisor I/O 虚拟化 / vDPA / vhost / SR-IOV passthrough**）；次领域：**半导体 / DPU**（DPU vDPA 实现）

## 2. 5 大主体类型

| # | 主体类型 | 角色 | 命中独立权 |
|---|---|---|---|
| **S1** | **Hypervisor / Host 软件厂** | Red Hat / VMware / Microsoft / Citrix；Linux kernel vDPA 框架；openEuler | Claim 19（Host 装置）|
| **S2** | **公有云 / 私有云部署方** | AWS / Azure / GCP / 阿里云 / 腾讯云 / 中国电信 / 中国移动 | Claim 1 / 11（方法）|
| **S3** | **服务器整机 / DPU 整机厂（OEM）** | 出厂含 vDPA / SR-IOV 整机产品 | Claim 20 / 21（计算节点 / 系统）|
| **S4** | **网卡 / DPU 芯片厂 + SDK** | NVIDIA / AMD/Pensando / Intel / Marvell；中国 DPU 创业（中科驭数 / 云豹 / 星云 / 大禹）| Claim 19 + 整机权间接 |
| **S5** | **Linux 内核上游 / vDPA 框架贡献者** | Linux kernel vhost / vfio / vDPA / DPDK 上游；Red Hat 主导 | 上游归因 |

## 3. 必跑参考清单

#### S1 — Hypervisor / Host 软件
- (a) 产业全景：
  - **Red Hat（IBM 子公司，NYSE: IBM）—— RHEL + OpenShift + RHV**
  - **VMware / Broadcom（NASDAQ: AVGO）—— vSphere ESXi**
  - **Microsoft（NASDAQ: MSFT）—— Hyper-V**
  - **Citrix（已私有化）—— Hypervisor**
  - **Canonical（一级未上市，发行 Ubuntu）**
  - **SUSE（私有化）**
  - **Oracle（NYSE: ORCL）—— Oracle VM Server / VirtualBox**
  - **Linux kernel vDPA 上游（CT-1 候选）**
  - **openEuler / 阿里 Anolis OS**
- (b) NASDAQ / NYSE: IBM / AVGO / MSFT / ORCL；港股 / A 股: 无直接发行版（中国 OS 发行多为 Linux fork）
- (c) Linux Foundation Networking / OPI / 各国信工委成员

#### S2 — 公有云 / 私有云
- 同 Patent 1：AWS / Azure / GCP / 阿里 / 腾讯 / 中国电信 / 中国移动 / 字节火山引擎

#### S3 + S4 — 整机 / DPU + 芯片
- NVIDIA BlueField + DOCA + mlx5_vdpa（**最直接的 vDPA 实现 — 上游 Linux kernel 主线 vDPA 由 Mellanox/NVIDIA + Red Hat 共同贡献**）
- AMD/Pensando + vDPA 实现
- Intel IPU + ifcvf vDPA driver（Intel 上游 vDPA driver）
- Marvell OCTEON + vDPA
- 中科驭数 / 云豹 / 星云 / 大禹 / Asterfusion + vDPA

#### S5 — Linux kernel + DPDK 上游
- Linux kernel vDPA 框架（drivers/vdpa/）
- Linux kernel vhost-vdpa driver（drivers/vhost/vdpa.c）
- DPDK rte_vdpa 框架
- QEMU virtio-net 后端 + vhost-vdpa 后端

## 4. 适用规则集

| 标志 | 是否激活 | 判定依据 |
|---|---|---|
| **R-OPENSOURCE** | ✅ TRUE | Linux kernel vDPA + DPDK + QEMU 全栈开源；最直接对应专利 |
| **R-CONFIG** | ✅ TRUE | vDPA 配置（vdpa-mgmtdev / vhost-vdpa devices / virtio-net interface）有具体配置参数 |
| **R-PARTNER** | ✅ TRUE | NVIDIA × Red Hat × VMware vDPA 联合白皮书丰富 |
| **R-PATENTWALL** | ✅ TRUE | G06F9/4555 主分类厚 |
| **R-PROCURE** | ✅ TRUE | NFV / 私有云集采涉及 |
| **R-STANDARD** | ⚠️ 部分 | virtio 1.x 是 OASIS standard；vDPA 是 Linux kernel framework，不直接构成 SEP |

## 5. 影响后续步骤

按 S1-S5 展开候选；**Linux kernel vDPA 上游 + Red Hat（CT-1）**作为核心技术候选独立 P0；**NVIDIA mlx5_vdpa（上游主要贡献者 + 商业 BlueField / ConnectX 卖家）独立 P0**。R-OPENSOURCE 双层激活；R-CONFIG 双引证。
