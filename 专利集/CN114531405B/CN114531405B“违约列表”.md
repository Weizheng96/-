### 4.2 现有技术 caveat（**关键**）

**Linux kernel bonding driver + mlx5 LAG offload 在 2017+ 已实现**，**早于专利申请日 2020-10-31**。如重合度 ≥ 60%，本专利**新颖性可能受影响 / 已授权专利可被无效宣告**。建议法务通过：
- (a) Linux kernel git log `--before=2020-10-31 -- drivers/net/ethernet/mellanox/mlx5/lag/`
- (b) Mellanox/NVIDIA 在 2017-2020 USPTO 同主分类同族专利
- (c) ConnectX-5 / 6 historical user manual（2017-2020）

来确认是否存在 ≥ 60% 重合的现有技术。如确认重合，本报告所有"公开资料不足强候选"档候选的可执行性都需重新审视。

### 4.3 Patent license / OIN caveat

本专利涉及 Linux kernel + OVS 上游主线技术——大量下游使用方覆盖整个云 / 网络生态。维权前法务必须核查：
- 华为是否是 OIN（Open Invention Network）成员？OIN 包含哪些与本专利相邻的核心专利？
- 华为是否对 Linux Foundation / OVS 项目做过任何 patent commitment / pledge？
- 华为在 ODCC / CCSA 标准化组织的 IPR 声明记录

如华为已做 patent pledge 或 OIN 承诺，则上游 OVS / Linux kernel + 商业发行版 + 商业 vendor 实施可能受 license 覆盖——影响"公开资料不足强候选"档的最终维权可行性。

### 4.4 法律免责声明

本报告输出**线索与证据链**，**不构成法律意见**。本专利目前法律状态为 **Active（已授权 2023-06-06）**，但存在**强现有技术 caveat**（Linux kernel bonding + mlx5 LAG 2017+ 已实现）+ patent license caveat 双重风险点；建议法务在维权前优先完成 (a) 现有技术深读 + (b) OIN / Linux Foundation patent commitment 核查。专利侵权最终判定由专业律师分析。
